Перелік основних команд: 
git config --global user.name 
git config - global user.email
git init - створення пустого репозиторію 
git status - відображення статусу репозиторію
git add - додати нові або змінені файли, щоб наступним кроком використати git commit
git commit - це знімок внесених змін у репозиторій 
git log - відображає історію комітів
git checkout - зміна гілки
git branch - відображення поточної гілки
git push - відправити зміни в репозиторій
git pull - отримання даних з віддаленого репозиторію 
git merge - об'єднання змін в одну гілку
git clone - клонування проекту в нову створену директорію
.gitignore файл - це певні файли, які будуть ігноруватись гітом

Advanced:
git revert - це команда, яку ми використовуємо, коли хочемо взяти попередній commitі додати його як новий commit, зберігаючи logнедоторканим.
git reset - це команда, яку ми використовуємо, коли хочемо перемістити репозиторій назад до попереднього commit
git rebase - це процес переміщення послідовності стосовно нового базового commit або їх обєднання 
git cherry-pick -  дозволяє інтегрувати вибрані окремі коміти з будь-якої гілки у поточну гілку HEAD.
git stash - бере змінений стан вашого робочого каталогу, змінені зареєстровані файли та проіндексовані зміни, і зберігає їх у сховищі незавершених змін, які можна в будь-який час змінити назад.

<p style="text-indent: 25px;">
    Aikido is a modern Japanese martial art that is split into many different styles. 
Aikido is now practiced in around <strong>140 countries</strong>. 
It was originally developed by <strong>Morihei Ueshiba</strong>, as a synthesis of his martial studies, 
philosophy and religious beliefs. Ueshiba's goal was to create an art that practitioners 
could use to defend themselves while also protecting their attackers from injury. 
Aikido is often translated as <strong>"the way of unifying (with) life energy"</strong> or as <strong>"the way of harmonious spirit"</strong>.
According to the founder's philosophy, the primary goal in the practice of aikido is to overcome oneself 
instead of cultivating violence or aggressiveness. 
</p>

<img src="https://aikidojournal.com/wp-content/uploads/2017/11/DSCF0464.jpg" alt="Exploration of Mars" width="450" height="300"> 
<figcaption><i>An aikido match</i></figcaption></figure>
<br>

<h3>Mental training </h3>
<p style="text-indent: 25px;">Aikido training is mental as well as physical, emphasizing the ability to relax the mind and body even under the stress of dangerous situations.
    This is necessary to enable the practitioner to perform the 'enter-and-blend' movements that underlie aikido techniques, wherein an attack is met with confidence and directness. 
Morihei Ueshiba once remarked that one "must be willing to receive 99% of an opponent's attack and stare death in the face" in order to execute techniques without hesitation. 
As a martial art concerned not only with fighting proficiency but with the betterment of daily life, this mental aspect is of key importance to aikido practitioners.</p>



<p><h3>Aikido's fundamental principles include:</h3></p>
<ul>
    <li>irimi (entering)</li>
    <li>atemi</li>
    <li>kokyu-ho (breathing control)</li>
    <li>sankaku-ho (triangular principle)</li>
    <li>tenkan (turning) movements that redirect the opponent's attack momentum.</li>
 </ul>
